AFFORDRAG-FACTORY: A Hierarchical Edge VLM and Knowledge Graph Framework for Ambiguous Multi-Robot Task Execution in Manufacturing

Yang Meng, Xiaoou Yang, John B. Frericks, Jiajia Xie, Tyler De Austria, Aarti Singh, Beshoy Morkos*
University of Georgia   |   Santa Clara University
ASME IDETC/CIE 2026
*Corresponding Author
Paper Code Video

AffordRAG-Factory converts ambiguous worker instructions and scene images into executable multi-robot Skill-Code commands through hierarchical reasoning over edge-side VLMs and knowledge graphs.

Abstract

High-Mix Low-Volume (HMLV) manufacturing often requires close worker involvement, but limited local computing resources and data compliance concerns make cloud-dependent embodied intelligence impractical for many small and medium-sized enterprises. At the same time, workers frequently issue ambiguous, effect-oriented instructions without explicitly specifying tools, actions, or constraint parameters, making end-to-end multimodal generation unstable in real deployment.

To address this challenge, we propose AffordRAG-Factory, a hierarchical embodied multi-robot task execution framework that combines the semantic understanding ability of edge-side small vision-language models (VLMs) with the structured retrieval, constraint filtering, and reasoning capability of knowledge graphs (KGs). The framework contains four layers: task intention analysis, tool affordance reasoning, action reasoning, and robot allocation, and supports cross-platform deployment through a unified Skill-Code protocol.

Experimental results on a benchmark of 35 ambiguous manufacturing instructions show that the framework achieves both high accuracy and low latency. Real-world case studies on a humanoid robot, a robotic arm, and a quadruped robot further demonstrate strong generalization and deployment feasibility across heterogeneous robot platforms.

Four-layer reasoning framework: task intention analysis, tool affordance reasoning, action reasoning, and robot allocation.

Humanoid grasping case study: the system infers WD-40 from an ambiguous maintenance request and executes tool grasping on a Unitree G1 EDU robot.

Robotic-arm defect inspection case study: the system generates a structured multi-view scanning plan for a Dobot E6 platform.

Quadruped navigation case study: the framework infers an electric drill request and dispatches a navigation task to a Unitree Go2 robot.
